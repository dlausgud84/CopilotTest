## 개요
이 문서는 사용자 관점에서 시스템 화면의 기본 사용 방법을 설명합니다.

## 로그인
- 로그인 페이지에서 계정을 입력해 인증 후 진입합니다.
- 세션이 만료되면 로그인 페이지로 이동합니다.

## 메뉴 관리
- 목록 조회, 검색, 정렬, 페이지 이동을 지원합니다.
- 수정/삭제 액션은 행의 마지막 컬럼에서 수행합니다.

## 문제 해결
- 화면이 비정상 동작하면 새로고침 후 재로그인합니다.
- 동일 문제가 반복되면 관리자에게 발생 시점과 화면 정보를 전달합니다.
