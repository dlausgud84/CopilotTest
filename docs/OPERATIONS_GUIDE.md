## 개요
개발/운영 환경에서 서비스 기동, 중지, 기본 점검 절차를 정의합니다.

## 기동/중지 순서
1. 백엔드 기동: `cd backend && ./gradlew :apps:main-app:bootRun`
2. 프론트엔드 기동: `cd frontend/noroo-mes-app && npm run dev`
3. 중지 시 프론트엔드/백엔드 순서로 종료

## 배포 점검
- 환경변수 및 프로파일 확인
- DB 연결 및 마이그레이션 확인
- 핵심 API 헬스체크 수행

## 장애 대응
